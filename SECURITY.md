# Security Policy

## 支持范围

仅维护当前主版本和最近一个稳定版本的公开 SDK 示例。

## 报告安全问题

请通过小狗模组平台公布的安全联系邮箱私下报告漏洞。报告中请包含：

- 受影响版本；
- 可复现步骤；
- 预期与实际结果；
- 可能影响；
- 不含真实用户隐私和生产密钥的最小示例。

请勿在公开 Issue 中发布：

- Project Secret、Token、私钥或数据库信息；
- 可直接利用的生产环境漏洞；
- 官方车辆内部变量、触发器或未公开接口；
- 未经授权获取的第三方源代码。

本仓库不接受以绕过付费、激活、授权或技术保护为目的的漏洞利用代码。
