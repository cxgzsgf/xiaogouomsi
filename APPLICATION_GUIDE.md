# 开发者接口申请指南

1. 登录小狗模组平台开发者接入中心。
2. 填写开发者、项目、车型、使用场景和所需能力。
3. 提交目标车辆权利或授权材料。
4. 等待身份、权限、安全和适配范围审核。
5. 审核通过后一次性领取开发项目凭据。
6. 将凭据保存在安全的本地开发环境，不要提交到 GitHub。
7. 使用本 SDK 实现自己的抽象适配器，并在车辆端执行安全检查。
8. 正式发布前重新提交适配包审核和签名。

## 禁止事项

- 访问、猜测或推导官方车辆私有接口和变量映射；
- 在源码、安装包明文配置或日志中泄露项目密钥；
- 未经授权修改或分发第三方车辆文件；
- 绕过车辆付费、激活、授权或技术保护；
- 把客户端申报的能力直接当作服务器授权；
- 在行驶状态下执行不安全的开门或高风险控制。
