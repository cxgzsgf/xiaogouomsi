# 架构概览

本 SDK 只公开接入，不公开平台代码。

```text
控制界面
  │
  │ HTTPS 开放接口
  ▼
平台鉴权与命令队列（闭源）
  │
  │ 标准命令，例如 lights.head.on
  ▼
VehicleAdapter（公开接口）
  │
  │ 车辆作者自行映射
  ▼
目标车辆自己的脚本或插件
```

## 设计原则

1. 平台只认识标准命令，不认识目标车辆内部变量。
2. 适配器只实现获批能力，不接受任意字符串执行。
3. 车辆端拥有最后安全决定权。
4. 匹配码用于建立车辆会话，不是永久控制密钥。
5. 示例代码不包含正式环境密钥或官方车辆内部验证系统逻辑。
